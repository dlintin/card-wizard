Overview

A simple but intractive jQuery wizard to replace the boring long forms or to make things more intractive

Idea

The main idea behind this resource is to replace the boring forms, somtimes user does not fill the form just becuase user finds it long enough to make him feel lazy. We splitted form fields into few groups and shown them in cards, with this user will see very less fields and it will feel more interesting with all the 3d animations and good UX

Usage

Basic Structure

We tried to keep things simple. Basic structure of "Card Wizard" consists of cards in an unorderd list with navigation. Navigation holds the Back Button, Next Button and Finish Button which is optional.

  1. card-front is in the front of all the cards
  2. card-middle is in the second place in the card stack
  3. card-back is the 3rd and last visible card
  4. card-out can be the extra cards you wants to place. These cards will be visible whenever they are on 1st, 2nd or on 3rd place in the card stack

Giving the styles

Setting the perspective to 500px to give it 3d feel and adding the z-index to all the cards to arrange them
